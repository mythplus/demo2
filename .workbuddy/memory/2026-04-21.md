- 修复 `webhook_service._load_webhooks()` `SyntaxError: name '_webhooks_cache' is used prior to global declaration`：P1-5 加 TTL 缓存时 `global _webhooks_cache, _webhooks_cache_expire` 被放到了第二个 `with _webhooks_cache_lock:` 块内（第 473 行），但函数内第 468 行已经先读过 `_webhooks_cache_expire`——Python 语法规则是 `global` 必须在函数内任何对该变量的引用**之前**声明，否则直接 SyntaxError。修正：把 `global` 行移到函数顶部（`_load_webhooks()` 首行）。`python -m py_compile` 验证通过。教训：读写同一个模块级变量的函数，`global` 统一放函数开头，不要拆到子块里。
